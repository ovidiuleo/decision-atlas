Decision Atlas: Phonological Analysis Tool
Decision Atlas is a professional-grade Streamlit application designed for Speech-Language Pathologists (SLPs) and students to analyze child phonology. It automates the detection of phonological processes, maps them to system-wide dimensions, and provides evidence-based clinical reasoning.

🚀 Key Features
Automated Detection: Identifies processes like Weak Syllable Deletion, Final Consonant Deletion, and Stopping based on phonetic input.

Systemic Mapping: Groups patterns into clinical dimensions such as Place Contrasts, Manner Contrasts, and Coda Stability.

Developmental Interpretation: Compares detected patterns against typical resolution ages (e.g., 36 months for Final Consonant Deletion) to highlight concerns or atypicality.

Educational Exploration: A searchable knowledge base of phonological processes, complete with descriptions, examples, and suggested therapy approaches like the Cycles or Minimal Pairs.

Case Export: Generates a professional .txt summary of observations and interpretations for clinical documentation.

🛠️ Installation & Local Run
If you wish to run this locally on your machine:

Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/decision-atlas.git
cd decision-atlas
Install dependencies:

Bash
pip install streamlit pandas
Run the app:

Bash
streamlit run app.py
📖 How to Use
Input & Observations: Enter the child's age and record your observations (Target vs. Produced).

Analysis: View the automatically detected patterns and how they relate to typical development.

System Map: Interact with the visual hierarchy of phonology to see the "big picture" of the child's speech system.

Export: Download a formatted summary for your clinical records.
